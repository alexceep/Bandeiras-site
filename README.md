# 🌍 Flag Wiki

Uma wiki interativa dedicada a **bandeiras de países**, combinando informação educativa com um sistema de quiz dinâmico para aprendizado.

---

## 📚 Sobre o projeto

O **Flags Wiki** é um site desenvolvido com o objetivo de apresentar informações sobre bandeiras de diversos países, permitindo que usuários explorem conteúdos e testem seus conhecimentos de forma interativa.

O projeto une:
- Conteúdo informativo estilo wiki  
- Interface visual organizada  
- Sistema de quiz funcional  

---

## 🌐 Acesse o site

👉 https://alexceep.github.io/Bandeiras-site/

---

## 🧠 Funcionalidades

- 🌍 Visualização de bandeiras de países  
- 📖 Conteúdo organizado em formato de wiki  
- 🎮 Quiz interativo sobre bandeiras  
- ⚡ Sistema dinâmico feito com JavaScript  
- 🎨 Interface simples e intuitiva  

---

## 🎮 Quiz interativo

O site conta com um sistema de quiz totalmente funcional, desenvolvido em **JavaScript**, permitindo:

- Perguntas dinâmicas  
- Interação em tempo real  
- Feedback imediato ao usuário  
- Experiência educativa e divertida
